Tree Regression Forecasting
In this project we were able to determine the funding needs of Ontario libraries based on the population of the communities they are in.
View Code
Project Details
Ontario provides funding to Ontario Public libraries every year. What determines how much funding a library will need? What would the funding requirements of a newly built library be? How much funding will libraries need next year?
The Ontario Library Survey data was analysed with the intention of determining the answers to these questions. After analysis it was found that population was the feature correlated with funding which could be used help answer the questions being asked. If we generate a model based on population which determines funding, then we would only need to know the population of the location where the new library was being built to determine it’s funding needs. We would only need to look at the forecast for Ontario’s population next year to know how much funding the libraries would need.
However simple linear regression returns a poor R2 when preformed on these two features. What next? Tree regression is the answer the model preformed much better and returned a R2 of 0.87.
To demonstrate the model funding was predicted a library in a city with a population of 100,000. The model predicts the funding given to a city with that population would be 166,475 and from the MAE we can expect that the actual value will be within plus or minus 17,281 of the predicted value.
